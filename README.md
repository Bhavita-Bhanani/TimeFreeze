# Time Freeze

A fast-paced browser survival game where movement becomes your greatest risk.

In **Time Freeze**, enemies relentlessly pursue the player. The catch? Time moves significantly faster when you move and slows down when you stand still. Every decision becomes a tradeoff between positioning and survival.

## Overview

Time Freeze is a lightweight browser game built entirely with vanilla web technologies.

The project was created as part of my **#75DaysOfCode** challenge to explore game development concepts, real-time rendering, and gameplay mechanics using HTML5 Canvas and JavaScript.

## Features

* Real-time gameplay using HTML5 Canvas
* Unique "Time Freeze" movement mechanic
* Dynamic enemy spawning system
* Survival timer and score tracking
* Keyboard support (WASD and Arrow Keys)
* Responsive fullscreen game canvas
* Game Over and restart system

## Gameplay

Your objective is simple:

1. Survive as long as possible.
2. Avoid incoming enemies.
3. Use the Time Freeze mechanic strategically.

### Controls

| Action     | Keys  |
| ---------- | ----- |
| Move Up    | W / ↑ |
| Move Down  | S / ↓ |
| Move Left  | A / ← |
| Move Right | D / → |
| Start Game | Space |
| Restart    | R     |
